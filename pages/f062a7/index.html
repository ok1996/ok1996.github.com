修改已提交的commit的注释

# 修改某次的commit的注释

例：修改倒数第16次commit的注释

git rebase -i HEAD~16
1

进入vim后，i进入编辑模式 将需要修改的那行的单词pick修改为edit,保存退出

# 修改注释

执行命令

git commit --amend
1

i进入编辑模式，然后修改注释，保存退出

# 执行命令

git rebase --continue
1

# 强制推送到远端

例：分支名称2022.1.x

git push  -f origin 2022.1.x
1

# 缺点

会将修改注释的提交时间自动修改为当前
