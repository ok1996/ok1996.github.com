安装Ubuntu

# 安装Ubuntu18.04

# 制作USB启动盘

下载

  • ubuntu-18.04.2-desktop-amd64.iso (opens new window)
  • rufus (opens new window)

制作启动盘，设备选择U盘、选择ISO镜像、点击开始

# 安装

U盘接入服务器，BIOS选择U盘启动

# 启动顺序

在BIOS选择U盘启动

# 安装引导界面

选择Install Ubuntu

# 安装简体中文

# 键盘布局

选择英语

# 安装软件

选择正常安装

# 用户密码

输入服务器用户名称及密码

# 初始化系统

进入服务器终端安装openssh vim

sudo apt install openssh-server vim -y
sudo vim /etc/ssh/sshd_config
修改以下配置：
-------------------------------
 LoginGraceTime 2m
 PermitRootLogin yes
 StrictModes yes
 #MaxAuthTries 6
 #MaxSessions 10
-------------------------------
sudo service ssh restart
