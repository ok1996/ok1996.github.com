# 首先启动mysql（主）和mysql（从）

这里使用docker-compose部署 设：本台服务器IP地址为192.168.123.2

部署mysql主从文件

链接：https://pan.baidu.com/s/1YDW6YALWZTRF0xtWkjwCzQ 
提取码：lmkf

2

数据库密码为i123456，在启动的时候需要对log文件及文件夹赋权

chmod -R 777 mysql/log
chmod -R 777 mysql/log/*
chmod -R 644 mysql/conf/my.cnf
chmod -R 777 mysqlread/log
chmod -R 777 mysqlread/log/*
chmod -R 644 mysqlread/conf/my.cnf

docker-compose up -d
